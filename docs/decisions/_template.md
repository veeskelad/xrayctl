## Context

What is the situation that calls for a decision? Cover the problem, the
constraints, what we already tried, and any non-obvious facts that make this
hard. Keep to facts; opinions go under Decision.

## Decision

What did we choose. State it in one or two sentences, then expand if the
choice has structure (multiple sub-decisions, a phased rollout, etc.).

## Consequences

What changes because of this decision.

- **Positive:** the wins we expect.
- **Neutral:** facts that follow but aren't wins or losses (new conventions,
  migrations, etc.).
- **Negative:** the costs we accept (debt, lock-in, complexity, deferred
  capability).

## Alternatives considered

Optional but recommended for non-trivial decisions. One short paragraph per
serious alternative, with the reason it lost.
